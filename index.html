🏸 Guide d'utilisation

📝 S'inscrire

  1. Entrez votre Nom / Prénom
  2. Cliquez sur "✅ Je participe !"
  3. Vérifiez que votre nom apparaît dans la liste

✏️ Modifier son inscription

  1. Cliquer sur ✏️ à côté de votre nom
  2. Modifier votre nom
  3. Cliquer sur "💾 Enregistrer"

🗑️ Se désinscrire

  1. Cliquer sur 🗑️ à côté de votre nom
  2. Confirmer dans la fenêtre qui s'affiche

💬 Partager via WhatsApp

Cliquer sur 💬 Partager dans l'en-tête pour envoyer le lien via WhatsApp avec un message pré-rédigé.

🖨️ Imprimer la liste

Cliquer sur 🖨️ Imprimer pour une impression propre avec uniquement la liste des participants.

🔥 Synchronisation temps réel

✅ Mises à jour instantanées — Tous les changements sont visibles par tous en temps réel, sans rafraîchir la page.

📊 Google Sheet

Le bouton Google Sheet est consultatif uniquement. Les inscriptions se font exclusivement depuis cette application.
